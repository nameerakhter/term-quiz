## What's Inside?

Test your knowledge with terminal-based quiz app.

## Getting Started

Make sure you have [Bun](https://bun.sh) installed on your system.

1. Clone this repo:

   ```bash
   git clone <https://github.com/nameerakhter/term-quiz>
   cd term-quiz
   ```

2. Install dependencies:

   ```bash
   bun install
   ```

3. Start the quiz:
   ```bash
   bun run index.ts
   ```
